# 🧰 Skills recomendadas de ClawHub para URA

| # | Skill | Slug | Función | Útil para URA porque... | Instalación |
|---|-------|------|---------|--------------------------|-------------|
| 1 | Find Skills | `find-skills` | Busca entre 13.000+ skills | Autonomía para URA | `npx clawhub@latest install find-skills` |
| 2 | Skill Vetter | `skill-vetter` | Escanea skills con VirusTotal | Seguridad ante malware | `npx clawhub@latest install skill-vetter` |
| 3 | Self-Improving Agent | `self-improving-agent` | Memoria de errores y aciertos | URA mejora sola | `npx clawhub@latest install self-improving-agent` |
| 4 | AgentBrowser | `agent-browser` | Navegador headless | Recetas, facturas, noticias | `npx clawhub@latest install agent-browser` |
| 5 | Composio | `composio` | Conecta 860+ herramientas (GitHub, Slack, Gmail) | Integración total sin auth | `npx clawhub@latest install composio` |
| 6 | N8N Workflow | `n8n` | Automatizaciones por chat | Marketing y contabilidad recurrentes | `npx clawhub@latest install n8n` |
| 7 | Exa Search | `exa-search` | Búsqueda indexada en documentación real | Investigación IA con fuentes reales | `npx clawhub@latest install exa-search` |
| 8 | AgentMail | `agentmail/core` | Email propio para URA | Enviar facturas y notificaciones | `npx clawhub@latest install agentmail/core` |
| 9 | Eleven Labs Agent | `elevenlabs` | Voz + failsafe telefónico | Si email falla, llama por teléfono | `npx clawhub@latest install elevenlabs` |
| 10 | Home Assistant | `home-assistant` | Casa inteligente 100% local | Cámaras, luces, ambiente desde URA | `npx clawhub@latest install home-assistant` |

> ⚠️ Verificar siempre con `skill-vetter` antes de instalar.
